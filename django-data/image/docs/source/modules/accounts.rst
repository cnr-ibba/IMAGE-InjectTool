
Accounts App
============

.. _accounts-app:

The accounts application extends the default `Django authentication <https://docs.djangoproject.com/en/2.2/topics/auth/>`_
and `Django registration redux <https://django-registration-redux.readthedocs.io/en/latest/>`_
in order to support IMAGE-InjectTool two-steps authentication with :ref:`Biosample registration <biosample-app>`
and tracking :py:class:`uid.models.Organization` objects.
