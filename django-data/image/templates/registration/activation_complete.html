Your registration is almost done

Your InjectTool account is now activated. Now you need to add an AAP profile in order to submit data to BioSamples. The AAP profile is a service provided by EMBL-EBI that manage identities and authorizations in order to submit data to EMBL-EBI archives. This service is managed by EMBL-EBI and InjectTool will manage and submit your data on your behalf. You can provide an existing AAP account or generate a new one using InjectTool. If you don't know what to choose, please select Generate a new AAP account button below.

Please note that we don't store any 3rd party passwords inside IMAGE-InjecTool system, those data will be managed by EMBL-EBI. Please keep track of your AAP credentials: InjectTool is not able to modify an AAP profile, if you lost your AAP credentials you will need to contact EBI.
